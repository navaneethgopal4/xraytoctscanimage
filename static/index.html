CycleGAN for Unpaired Image-to-Image Translation

Medical Imaging Domain Mapping

Chest radiographs (2D X-rays) and axial CT slices represent drastically different imaging modalities. While X-rays are projection images displaying overlapping thoracic structures, CT slices capture cross-sectional anatomical slices.

SynthoCT utilizes a CycleGAN framework, enabling unsupervised translation without matched image pairs. It maps texture domains while conserving general structural configurations.

Deep Learning Architecture

  • Generators: A 9-block ResNet framework that downsamples, bottleneck-processes, and upsamples inputs, ensuring fine-grained, localized texture preservation.
  • Discriminators: A 70x70 PatchGAN network which evaluates overlapping $70\times 70$ pixel blocks for realistic medical texture mapping.
  • Loss Formulations: Integrates Least Squares GAN loss (LSGAN), L1 Cycle-Consistency loss (Weight: 10.0), and Identity Mapping loss (Weight: 5.0).
